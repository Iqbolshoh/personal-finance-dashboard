# 💰 FinTrack — Personal Finance Dashboard

A modern, easy-to-use web app for tracking your personal finances. Monitor your income and expenses, and visualize spending by category with interactive charts.

---

## ✨ Features

- 📊 **Balance, income & expense summary** — calculated in real time
- ➕ **Add transactions** — with description, amount, type, and category
- 🗑️ **Delete transactions** — with a single click
- 🍩 **Category breakdown chart** — visual analysis powered by Chart.js
- 💾 **LocalStorage** — your data is saved in the browser, even after closing the page
- 📱 **Responsive design** — looks great on both mobile and desktop
- 🎨 **Modern dark UI** — with clean, professional SVG icons

## 🚀 Getting Started

1. Clone the repository:
```bash
git clone https://github.com/Iqbolshoh/fintrack.git
cd fintrack
```

2. Open `index.html` in your browser — no installation needed!
